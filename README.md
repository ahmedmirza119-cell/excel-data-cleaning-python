# 📊 Excel Data Cleaning with Python

Ek real-world data cleaning project jisme messy sales data ko 
Python aur Pandas ki madad se clean kiya gaya hai.

## 🔍 Kya Problems Thi Data Mein?
- Extra spaces aur wrong casing (Sara KHAN, LAHORE)
- Duplicate rows
- Invalid emails aur phone numbers
- Missing values (NaN)
- Inconsistent date formats
- Wrong data types (numbers as text)
- Negative sales values

## 🛠️ Tools Used
- Python 3
- Pandas
- OpenPyXL
- Regex (re)

## 📁 Files
| File | Description |
|------|-------------|
| `messy_sales_data.xlsx` | Original dirty data |
| `clean_data.py` | Python cleaning script |
| `clean_sales_data.xlsx` | Final clean output |

## 🚀 Kaise Chalayein?
```bash
pip install pandas openpyxl
python clean_data.py
```
